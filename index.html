DockerDesk: Un'applicazione .NET per la Gestione di Ambienti Docker

DockerDesk è un'applicazione sviluppata in .NET Framework 4.7.2, progettata per offrire un controllo completo e versatile degli ambienti Docker sia in locale che su macchine remote. Utilizzando riferimenti a DLL di terze parti come Newtonsoft.Json per la gestione dei dati in formato JSON e Renci.SshNet per le connessioni SSH sicure, DockerDesk fornisce un'interfaccia utente grafica intuitiva e una serie di funzionalità avanzate per la gestione efficiente dei container, delle immagini, dei volumi e delle reti Docker.

Funzionalità Principali

  • Connessione e monitoraggio dello stato di Docker in locale e su macchine remote.
  • Gestione delle immagini Docker: caricamento, creazione ed eliminazione in ambiente locale e remoto.
  • Controllo completo dei container Docker: visualizzazione, esecuzione, rimozione e inspezione, sia localmente che a distanza.
  • Gestione avanzata dei volumi e delle reti Docker, consentendo operazioni di creazione, rimozione e configurazione.
  • Utilizzo di variabili d'ambiente per personalizzare i container Docker, con supporto alla lettura e scrittura di file JSON.
  • Interfaccia utente grafica intuitiva per un'interazione facilitata e un'esperienza utente migliorata.
  • Trasferimento e gestione di file tra l'host locale e la macchina remota attraverso connessioni SSH sicure.

DockerDesk Sessione Remota

Funzionalità di DockerDesk

  • Connessione SSH per gestire Docker su macchine Linux remote.
  • Caricamento e visualizzazione di immagini Docker dalla macchina remota.
  • Creazione ed eliminazione di immagini Docker sulla macchina remota.
  • Caricamento e visualizzazione di container Docker dalla macchina remota.
  • Rimozione ed esecuzione di container Docker sulla macchina remota con configurazioni avanzate.
  • Inspezione dettagliata dei container Docker sulla macchina remota.
  • Gestione dei volumi Docker sulla macchina remota: caricamento, creazione e rimozione.
  • Gestione delle reti Docker sulla macchina remota: caricamento, creazione, rimozione, connessione e disconnessione dei container.
  • Caricamento e visualizzazione delle variabili d'ambiente dei container Docker sulla macchina remota.
  • Trasferimento e decompressione di file sulla macchina remota via SSH.
  • Interfaccia grafica per la gestione della directory di lavoro locale e remota.
  • Lettura e scrittura di file JSON per gestire le variabili d'ambiente dei container Docker.
